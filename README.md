# Renal-Health-Prediction
End-to-end Machine Learning project for renal health prediction using Decision Tree Classification. Includes data preprocessing, feature engineering, model training, evaluation and kidney disease risk prediction based on patient medical attributes such as blood pressure, RBC count, serum urea, and hemoglobin.
## Features
- Data preprocessing and cleaning
- Feature engineering
- Decision Tree Classification
- Kidney disease risk prediction
- User-friendly web interface
- Real-time prediction using patient inputs
-  ## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript
- ## Dataset Attributes
- Age
- Blood Pressure (BP)
- Red Blood Cell Count (RBC)
- Serum Urea
- Hemoglobin
- Other clinical parameters
